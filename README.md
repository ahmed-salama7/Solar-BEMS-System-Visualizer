Solar Panel–Based Battery Energy Management System
Project by: Ahmed Salama — Istanbul Arel University

 Overview:
This repository contains the hardware documentation, Bill of Materials (BOM), and the interactive digital schematic visualizer for a 3S Li-ion Solar BEMS prototype. The system is designed to manage energy flow from a 12V 10W PV panel into a 3-cell battery pack.
